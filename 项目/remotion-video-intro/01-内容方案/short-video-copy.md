### 正文

跟 Claude 说句话，它就把视频项目搭好了。

这个 Skill 叫 remotion-video，装到 Claude Code 里，说"用代码做视频"就能触发。React 组件写场景、AI 自动配音、字幕动画一步到位。

几个值得试的点：

⚛️ **React 写视频**
每个场景一个组件，Remotion 原生支持

🔊 **双引擎配音**
Edge TTS 免费开搞，MiniMax 可以克隆声音

📝 **字幕动画全自动**
时间轴同步，不用手动对帧

```
git clone https://github.com/wshuyi/remotion-video-skill.git
cp -r remotion-video-skill ~/.claude/skills/
```
