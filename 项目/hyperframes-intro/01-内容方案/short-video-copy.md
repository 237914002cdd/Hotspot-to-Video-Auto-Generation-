### 正文

做视频不一定要学剪辑软件。

这个新工具叫 HyperFrames，HeyGen 刚开源的。思路很简单——你写 HTML 加 CSS 动画，它直接渲染成 MP4。

我试了一下，25 秒的短视频从写代码到出片，半小时搞定。

几个值得关注的点：

🔓 Apache 2.0 开源
商用不收钱，不存在什么 "免费之后要付费" 的套路

🤖 AI 写就行
让 Claude 写 HTML，你负责确认效果，真的不用自己敲代码

⚡ 三行命令
```
npx hyperframes init
cd my-video && npm install
npx hyperframes render
```

和 Remotion 比，优势是不用学 React。纯 HTML 就能跑，门槛低不少。

适合谁：
想做技术科普视频的开发者
需要批量做营销短片的运营
想试试 AI 做视频的人

GitHub 搜 hyperframes，直接上手。
