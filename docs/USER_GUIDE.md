# 用户快速上手指南

> 目标读者：非开发人员。你不需要懂代码。
> 预计时间：10 分钟从零到出第一个视频。

---

## 前置准备

### 1. 安装 Node.js

1. 打开浏览器，访问 https://nodejs.org
2. 下载 **LTS 版本**（左侧按钮，推荐 18 或 20）
3. 双击安装包，一路点"下一步"直到完成
4. 验证安装：
   - Windows：按 `Win+R` → 输入 `cmd` → 回车
   - 在黑色窗口输入 `node --version` → 回车
   - 如果看到 `v18.x.x` 之类的输出，说明安装成功

### 2. 下载本项目

- 从 GitHub 仓库下载 ZIP，解压到你的电脑
- 或者如果你会用 Git：`git clone [仓库地址]`

## 启动看板（5 步）

### 第 1 步：打开终端

```
AI视频工作流/
├── 看板/        ← 进入这个目录
├── 项目/        ← 你的视频项目在这里
├── 模板/        ← 封面模板
└── ...
```

在文件管理器中进入 `AI视频工作流/看板/` 文件夹，按住 `Shift` 键 → 右键空白处 → 选择"在此处打开 PowerShell 窗口"。

### 第 2 步：安装依赖

在终端中输入：

```bash
npm install
```

等待几秒，看到 `added X packages` 就完成了。

### 第 3 步：启动服务

```bash
node server.js
```

看到如下画面说明启动成功：

```
╔══════════════════════════════════════════╗
║  内容决策看板 · Content Dashboard        ║
║  http://localhost:3456                 ║
╚══════════════════════════════════════════╝
```

> **注意**：这个窗口不要关，关了就停止运行了。

### 第 4 步：打开看板

打开浏览器（Chrome / Edge 都行），地址栏输入：

```
http://localhost:3456
```

你会看到看板首页，包含热点推荐、项目资产总览、快速录入等板块。

### 第 5 步：做一个视频

#### ① 选题（可选）
- 切换到"热点简报"页面，看自动抓取的 GitHub 热门话题
- 切换到"选题矩阵"页面，给选题打分

#### ② 创建项目
在"选题矩阵"中点击一个选中的话题 → 看板会自动创建项目文件夹

或者在"总览"页面右侧的"项目资产管理"区域，查看已有项目列表

#### ③ 生成剧本
找到项目卡片，点击蓝色 `✨ 生成剧本` 按钮
- 弹窗中会显示 HTML 代码和社交文案
- 点击"保存到项目"按钮

#### ④ 渲染视频
保存后，蓝色按钮变为 `🎬 开始渲染`
- 点击它 → 看板开始后台渲染
- 渲染过程中你可以继续浏览其他页面
- 完成后按钮变为 `🔄 重新渲染`

#### ⑤ 做封面
- 点击 `🎨 生成封面` → 看板自动根据项目内容生成封面
- 如需修改，点击"编辑封面"调整文字

#### ⑥ 存档
- 有视频 + 有封面后，点击 `📦 一键归档`
- 看板自动搬运文件、写 README、记录生产日志

## 常见问题

**Q: 启动后浏览器显示"无法访问此网站"？**
A: 确认终端窗口没关，地址输入的是 `http://localhost:3456`（不是 https）。

**Q: 点击按钮没反应？**
A: 先确认服务器是否还在运行（终端窗口有没有报错）。如果服务器崩溃了，重启 `node server.js`。

**Q: 封面和视频是 AI 自动生成的吗？**
A: 封面是从你的项目内容自动提取文字填充模板；视频渲染依赖 HyperFrames 框架，渲染方式由你选择的 HTML 模板决定。

**Q: 如何关闭看板？**
A: 在终端窗口按 `Ctrl+C`，或者直接关掉终端窗口。
